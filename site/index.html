Base Grid

Deploy Now works on CSS Grids. Your standard grid in this framework is a 4 by 4 grid represented below.

If your design requires more than what can be accomplished in a 4x4 Grid simply break and start a new 4x4 grid.

The grid elements can be redesigned each by adding a custom class in the style.scss file.

For wire framing purposes the 'Template' class will add dashes around your cells.
